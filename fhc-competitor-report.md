## Executive Summary

目前市场上针对 **自雇客户 / SME 贷款健康检查** 的免费工具较少。大多数工具：
- 面向 **个人住房贷款** (residential)
- 需要 **付费订阅** 或为 **获客漏斗** 服务
- 缺乏 **专业咨询级别** 的报告输出

**Oney & Co FHC** 的差异化机会：
1. 专注 **自雇 / SME** 客户群体
2. **完全免费**，降低使用门槛
3. **麦肯锡咨询风格** 的专业报告
4. 报告可直接发给 Banker/Broker 进行 Review

## 竞品分析

### 1. Nav.com (US)
**定位:** 小企业融资匹配平台

| 项目 | 详情 |
|------|------|
| **免费功能** | 信用评分监测、贷款匹配 |
| **付费功能** | $29.99/月 Premium - 更多贷款选项、身份保护 |
| **优势** | 贷款类型全面、快速匹配 |
| **劣势** | 美国市场、无专业报告输出 |

### 2. Athena Home Loans (AU)
**定位:** 低费用住房贷款

| 项目 | 详情 |
|------|------|
| **免费功能** | 利率比较、还款计算器 |
| **收费** | 无申请费、无持续费用 |
| **优势** | 透明定价、AcceleRATES 自动降息 |
| **劣势** | 仅限住房贷款、无 SME 产品 |

### 3. Canstar / RateCity (AU)
**定位:** 金融产品比较平台

| 项目 | 详情 |
|------|------|
| **免费功能** | 利率比较、计算器 |
| **收费** | 免费（广告模式） |
| **优势** | 覆盖面广、品牌信任度高 |
| **劣势** | 无个性化报告、无 SME 专项工具 |

### 4. SCORE.org (US)
**定位:** 小企业导师/资源平台

| 项目 | 详情 |
|------|------|
| **免费功能** | 商业健康评估问卷、导师匹配 |
| **收费** | 免费（政府资助） |
| **优势** | 全面的商业评估框架 |
| **劣势** | 美国市场、无贷款专项分析 |

### 5. 银行自有工具
**例如:** NAB Business Lending Calculator, CBA Business Health Check

| 项目 | 详情 |
|------|------|
| **免费功能** | 基础计算器、还款估算 |
| **优势** | 品牌信任、直接对接贷款 |
| **劣势** | 仅推销自家产品、无中立分析 |
